## 2. إعداد بيئة التسجيل

قبل الضغط على REC:

1. **متصفّح نظيف:** افتح Chrome في وضع Incognito — تختفي الـbookmarks والـextensions
2. **حساب تجريبي:** سجّل دخول بحساب `owner@demo.qit3ati.com` (بيانات تجريبية فقط — لا تكشف بيانات عميل حقيقي)
3. **اختفِ الـnotifications:** Do Not Disturb mode على macOS أو Focus Assist على Windows
4. **حجم النافذة:** 1920×1080 (أكبر زر من Chrome → عيّن window إلى size هذا)
5. **التكبير:** Cmd+0 لإعادة الـzoom للـdefault. الـUI يجب أن يكون مقروءاً ولكن غير مكبّر
6. **اللغة:** عربية للسوق الأردني، إنجليزية للأسواق الخليجية (سجّل نسخة لكل)
7. **تنظيف الـcursor:** Cursor Highlighter (Mac: Cursor Highlight ext، Win: ZoomIt) لرؤية أوضح
8. **اختبار الصوت:** سجّل 5 ثوان تجريبية، شغّلها بسماعات — تأكّد لا فيه echo

## 4. خطوات التسجيل والمونتاج

### الجلسة المثالية (يوم عمل واحد، 8 ساعات)

**ساعة 0–1: التحضير**
- اقرأ كل السيناريوهات الستة مرّة كاملة
- جهّز البيانات التجريبية (عملاء + أصناف + موردين دميين)
- جرّب 5 دقائق record + playback لتأكّد الصوت + الفيديو ok

**ساعة 1–4: التسجيل (3 ساعات)**
- 30 دقيقة لكل فيديو (record + retake)
- بين كل فيديو خذ 5 دقائق راحة + ماء + إعادة ضبط الـscene

**ساعة 4–5: راحة + غداء**

**ساعة 5–7: المونتاج (2 ساعات)**
- لكل فيديو: 20 دقيقة (قص الأخطاء + intro/outro + موسيقى خفيفة)
- استورد intro template من Canva + outro

**ساعة 7–8: النشر**
- ارفع على YouTube (Unlisted)
- اكتب title + description + tags لكل فيديو
- انسخ الـ11-character video ID لكل واحد

## 5. ربط الفيديوهات بالنظام

بعد رفع كل فيديو على YouTube:

1. افتح `apps/web/src/pages/help/HelpCenterPage.tsx`
2. ابحث عن `const VIDEOS: VideoItem[]`
3. الصق الـYouTube ID في الـ`youtubeId` لكل فيديو:

```ts
const VIDEOS: VideoItem[] = [
  { title: 'البدء السريع — 5 دقائق', duration: '5:23', youtubeId: 'dQw4w9WgXcQ' },
  // ... وهكذا
];
```

4. احفظ + commit + push → الفيديوهات تظهر تلقائياً مع الـthumbnail
5. النقر على أيّ بطاقة يفتح player داخل النظام (لا يطلع للـYouTube)

**لمن لا يريد YouTube:** استعمل `vimeoId` بدلاً من `youtubeId` — نفس الـsupport.

## 6. أفضل الممارسات

✅ **الـHook في أول 10 ثوان**: لو الـviewer لم يفهم القيمة، سيُغلق  
✅ **خطّ صوت موسيقى ناعم** (-20 dB) — يخلق إحساس احترافي  
✅ **Captions بالعربية والإنجليزية** — YouTube auto-generates، عدّلها يدوياً  
✅ **اسم الـvideo descriptive**: "قطعتي - إصدار فاتورة في 30 ثانية" بدلاً من "Tutorial 2"  
✅ **CTA في النهاية**: "اشترك للحلقات القادمة" + رابط للنظام  

❌ تجنّب: تسجيل في 4K (حجم ضخم بلا فائدة — 1080p كافية)  
❌ لا تستعمل Comic Sans أو خطوط مزخرفة في الـoverlays  
❌ لا تذكر أسعار محدّدة (تتغيّر) — أحل للموقع للأسعار الحالية
